Spline Transformations - Rendering OpenGL

Rendering a spline shape with a dedicated graphics engine written in Python. The scene graph architecture allows to create a custom spline-shape geometry and then locally transform the entire object with the disered operations. All matrix and vector operations are abstracted by the framework, allowing to translate, rotate and scale from well-defined methods. The stored model matrices are prepared to work with global and local transformations. Read the following post to deeply understand how matrix transformations works: Matrix Transformations for Rendering 3D Objects
